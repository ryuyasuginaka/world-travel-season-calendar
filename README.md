# 🧭 SEASON ATLAS — 世界旅行 ベストシーズンカレンダー

世界62地域の観光ベストシーズンを月別に一覧できるシングルページのWebアプリ。
「行きたい月」からも「行きたい場所」からも旅行先を探せます。

**公開URL: https://ryuyasuginaka.github.io/world-travel-season-calendar/**

ビルド不要のプレーンHTML/CSS/JS構成。ブラウザで開くだけで動きます（地図のみLeaflet CDNを使用）。

```
index.html      # マークアップ
css/style.css   # 旅行誌スタイルのテーマ
js/data.js      # 62地域データ（評価・理由・座標・物価Tier・航空券目安）+ 持ち物リスト
js/app.js       # 全ロジック
assets/hero.jpg # ヒーロー写真
```

## デザイン

フォトジャーナリスト旅行誌スタイル:

- **カラー**: オフホワイト背景 + ディープネイビー + テラコッタ + ゴールデンイエロー
- **タイポグラフィ**: Playfair Display（見出し）+ Montserrat（ラベル）+ Source Serif 4（本文）+ Bebas Neue（数字）
- **レイアウト**: 全画面ヒーロー写真（パララックス）、マガジン風セクションヘッダー、スクロール連動アニメーション

## 機能

- **行き先ルーレット** 🎰 — 行きたい月とエリアを選んでスピンすると、その条件で「◎ベストシーズン」の地域からスロット演出でランダムに行き先が決まる。結果から詳細表示・お気に入り追加も可能
- **ルートマップ** 🗺️ — 62地域を世界地図（Leaflet）にプロット。マーカー色は選択月の評価と連動し、お気に入りは追加順に周遊ルート線で結ばれる
- **費用シミュレーター** 💰 — 目的地×日数×スタイル（節約⇔快適）で旅費の概算を試算。現地物価Tierと東京発航空券相場から7カテゴリの内訳を算出
- **持ち物リスト** 🎒 — カテゴリ別チェックリスト＋気候別追加装備（暑い/寒い/雨季）。チェック状態はlocalStorageに保存
- **月別カレンダー** — 62地域 × 12ヶ月を ◎（ベスト）/ △（まずまず）/ ×（避ける）で表示。セルにホバー / タップで理由（気候・イベント）を表示
- **地域の詳細ビュー** — 地域名をクリックすると、12ヶ月の評価と理由を一覧するモーダルを表示
- **お気に入り・比較** — 各行の ★ で地域をピン留めし、ページ上部の比較テーブルに並べて表示（localStorageに保存され次回も維持）
- **地域フィルタ** — 9大エリア（東アジア〜オセアニア）で絞り込み
- **月ハイライト** — 月を選ぶと該当列を強調（初期表示は今月）。「◎のみ表示」でその月がベストシーズンの地域だけに絞り込み
- **おすすめパネル** — 選択した月がベストシーズンの地域を一覧表示。クリックで該当行へジャンプ
- **目的地検索** — 地域名のインクリメンタル検索
- **URL共有** — フィルタ状態がURLハッシュに同期（例: `#r=europe&m=10&best=1`）。そのままブックマーク・共有可能
- **モバイル対応** — 表の先頭列固定、スクロール時のフィルタバー自動コンパクト化、タップで開くツールチップ

## データの編集・追加

すべてのデータは `js/data.js` の `DATA` 配列にあります。1地域 = 1行:

```js
{ r: "east-asia", n: "🇯🇵 日本（本州）", c: 3, f: 2, g: [35.68, 139.77], m: [[1,"寒冷・降雪"], /* …12ヶ月分 */ ] },
```

- `r`: 地域キー（`REGIONS` 配列参照）
- `n`: 表示名（国旗絵文字 + 名前）
- `c`: 現地物価Tier（1=安い 〜 5=高い、費用シミュレーター用）
- `f`: 東京発 往復航空券の目安（万円）
- `g`: 緯度・経度（ルートマップ用）
- `m`: 12ヶ月分の `[評価, 理由]`。評価は `3`=◎ベスト / `2`=△まずまず / `1`=×避ける

行を追加するだけで表・統計・おすすめ・ルーレット・地図・費用計算すべてに自動反映されます。
持ち物リストは同ファイルの `PACKING` / `PACKING_CLIMATE` で編集できます。

## 免責・クレジット

このカレンダーは一般的な気候傾向に基づいています。実際の天候は年によって異なるため、旅行前に最新の現地情報をご確認ください。
